# S Niveditha Krishna — Developer Portfolio

A modern, responsive developer portfolio built with **React 19**, **TanStack Start (Vite)**, **Tailwind CSS v4**, and **Framer Motion**.

> Note: This project uses TanStack Start (a React full-stack framework on Vite) rather than Next.js. The DX is similar — file-based routing under `src/routes/`, SSR support, and a single `npm run dev` to start.

## ✨ Features

- ⚡ Smooth scroll navigation with a clean **Menu dropdown**
- 🌗 Dark mode toggle (persisted to localStorage)
- 🎨 Modern design system (OKLCH tokens, Sora + Inter + JetBrains Mono)
- 📱 Fully responsive design
- 🎬 Framer Motion animations (subtle & clean)
- 📝 Contact form with validation
- 📄 Resume download support (place `resume.pdf` in `public/`)
